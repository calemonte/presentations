How to Make Awesome Presentations with Reveal.js

What is Reveal.js?

A (free) framework developed by Hakim El Hattab for easily creating beautiful presentations using HTML.
Why use Reveal.js?

Embed external websites in your slides

Display videos, gifs, and images as backgrounds

Produce non-linear pathways through content

Create custom styles and themes

Export your presentation for easy hosting on the web

Interesting features like multiplexing

It's open source and extremely customizable

Folder structure

Open the entire reveal.js folder in your code editor

The reveal.js folder overview

  • index.html: Main file that you'll edit to add slides
  • css: Core styles without which the project does not function (also contains other default themes)
  • js: Like CSS but for JavaScript
  • plugin: Components that have been developed as extensions to reveal.js
  • lib: All other third party assets (JavaScript, CSS, fonts)

Everything else you can ignore 😎

                    Reveal.initialize({
                      dependencies: [
                        { src: 'plugin/markdown/marked.js' },
                        { src: 'plugin/markdown/markdown.js' },
                        { src: 'plugin/notes/notes.js', async: true },
                        { src: 'plugin/highlight/highlight.js', async: true }
                      ],

                      autoSlide: 5000,
                      progress: false,
                      transitionSpeed: 'fast',
                      backgroundTransition: 'zoom',
                    });
                </script>

PDF export

You must have a server running for this to work!*
  • Open your presentation with print-pdf included in the query string (i.e., http://localhost:8000/?print-pdf).
  • If you want to include speaker notes in your export, you can append showNotes=true to the query string (i.e., http://localhost:8000/?print-pdf&showNotes=true).

* More on this later!
